# healthcare-cost-analysis
An end-to-end data analytics project using SQL, Excel Pivot Tables, and Power BI to analyze healthcare insurance charges and identify key cost drivers like smoking and BMI.
# 🏥 Healthcare Insurance Cost Driver Analysis

# Project Overview
This project explores the primary drivers of healthcare insurance charges using SQL, Microsoft Excel, and Power BI. The goal was to identify key demographic and risk factors that disproportionately impact insurance costs.

 Key Findings
1. **The Smoking Multiplier:** Smoking status is the single largest cost driver. Non-smokers average **R8,434**, whereas smokers average **R32,050** (~4x higher).
2. **The "BMI + Smoker" Double Whammy:** Being obese increases costs moderately for non-smokers (R8,843), but for smokers, average costs skyrocket to **R41,558** (over 5x higher than normal non-smokers).
3. **Age Progression:** Insurance charges increase steadily across all age groups, but a 20-year-old smoker still incurs higher average charges than a 60-year-old non-smoker.

 Tools & Technologies Used
* **SQL (MySQL):** Data exploration, duplicate identification & removal, metric aggregations.
* **Microsoft Excel:** Data transformation, conditional logic (`IF` groupings), Pivot Tables, and Pivot Charts.
* **Power BI Desktop:** Dynamic dashboard design, DAX measures, data visualization.
